Homework 1

Resumen: hay que editar el perfil de GitHub, añadir un par de fortalezas y habilidades. Después descargue Anaconda y cree una carpeta en mi computadora llamada Github para todas las tareas de la materia. Procedimiento a descargar la herramienta de git para poder realizar las tareas necesarias para la asignación. Clone la el site web que se nos proporciono en la tarea, copie el link que nos proporcionó GitHub y lo pegue en la terminal de la computadora con el comando "git clone + el link". Después en TextEdit modifique el texto de la página web y luego procedí a guardarla. Por último después de terminar de modificar la página web realizare la ultima asignación de la tarea.
